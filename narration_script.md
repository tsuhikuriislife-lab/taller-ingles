Hi! Today I’m presenting my Number Guesser game. I built this to practice Python and my English. It’s a game where you try to find a secret number within a range you choose.

When the game starts, you see a menu. You can start a new game or exit. If you play, the code asks you for a start and an end number. I used a try-except block here so the program doesn't crash if you type a letter by mistake.

Once the range is set, the computer picks a random number. Now, you start guessing. If you guess wrong, the game shows you a list of all your previous guesses so you don’t repeat them. It also counts how many attempts you are making.

When you finally guess the right number, the game says "Congratulations!" and shows you the odds of your win—for example, a 1 in 10 chance. You can then go back to the menu to play again or leave.

I really enjoyed making this because it uses the basics of programming: loops, variables, and math. It’s simple, but it works perfectly. Thanks for watching!
