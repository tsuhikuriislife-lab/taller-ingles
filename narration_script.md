Today i'm going to talk to you about the project I made.

It is a fishing simulator that includes a handful of lakes to choose, different types of fish with various rarities and a weight randomizer.

It's very simple, only using loops and conditionals, and some other bland skills like inputs and prints.

This proyect was made from a practice I had while talking with my partner, because of that, some things are references to our life.

It was very fun making this come to life, and I'm very proud of how it turned out.

Let's talk a little more in depth about the game's functionality.

The game follows this simple path.

Start, the game gives you a short explanation about what it can do.
Then, the menu asks you if you want to play or not. If not, the game closes.
If you wanted to play, you then get to decide which lake to fish on.
The loop starts and you get to catch cool fish like the Golden fish, and the Ponyo fish, based on Ponyo from Hayao Miyasaki anime movies!
Every time you catch a fish, you can decide to go back and select to go to another lake.

The game is very simple in terms on functionality, but it has a lot of potential and very cool ideas to be added!
